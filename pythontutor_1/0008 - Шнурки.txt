# Условие
# Обувная фабрика собирается начать выпуск элитной модели ботинок. 
# Дырочки для шнуровки будут расположены в два ряда, расстояние между рядами равно aa, 
# а расстояние между дырочками в ряду bb. Количество дырочек в каждом ряду равно NN. 
# Шнуровка должна происходить элитным способом “наверх, по горизонтали в другой ряд, 
# наверх, по горизонтали и т.д.” (см. рисунок). 
# Кроме того, чтобы шнурки можно было завязать элитным бантиком, 
# длина свободного конца шнурка должна быть ll. 
#
# Какова должна быть длина шнурка для этих ботинок?
#
# Программа получает на вход четыре натуральных числа aa, bb, ll и NN - именно в таком порядке - и должна вывести одно число - искомую длину шнурка.


a = int(input())
b = int(input())
l = int(input())
N = int(input())

print((a+b)*2*(N-1) + a + l*2)
